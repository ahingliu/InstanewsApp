## PROJECT 3 - [INSTANEWS APP](https://instanewsapp.netlify.com//)

This is a one-page, responsive website (mobile first approach) that allows a user to filter the top news story categories via the New York Times API.

- CSS3
- SCSS
- JavaScript
- jQuery
- Ajax
- Gulp
